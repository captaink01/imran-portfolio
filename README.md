# Imran Muhammad Hamza — Portfolio

A personal portfolio built with **Angular 18** featuring 4 pages, Angular Router, and responsive design.

## Pages
- **Home** — Hero section with animated skills ticker
- **About** — Bio, skills grid, and career timeline
- **Projects** — Filterable project cards
- **Contact** — Contact info + enquiry form

## Tech Stack
- Angular 18 (standalone components)
- SCSS with CSS variables
- Angular Router with hash/history routing
- Deployed on GitHub Pages

## Run Locally
```bash
npm install
ng serve
```

## Deploy to GitHub Pages
```bash
chmod +x deploy.sh
./deploy.sh
```
Then push to GitHub and enable Pages in repo Settings.
